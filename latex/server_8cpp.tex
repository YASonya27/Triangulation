\doxysubsection{Detailed Description}
Сервер для обработки триангуляции данных, полученных от клиента 

Сервер принимает от клиента количество точек, генерирует случайные точки, выполняет их триангуляцию и отправляет результат обратно клиенту. Использует Winsock для сетевого взаимодействия.
